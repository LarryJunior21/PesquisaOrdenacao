# Trabalho Bimestral de Pesquisa e Ordenação

## Desenvolvido em Java usando NetBeans

_Metodos Utilizados:_
* Inserção Direta e Inserção Binária;
* Seleção Direta;
* Bolha e Shake;
* Shell;
* Heap;
* Quick (com e sem pivô);
* Fusão Direta (Merge) (duas implementações).
* Counting;
* Bucket;
* Radix;
* Comb;
* Gnome;
* Tim.
